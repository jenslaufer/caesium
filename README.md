# Caesium Systems — site

Static one-page company site for **Caesium Systems** (precision time & deterministic
low-latency infrastructure for capital markets). Pure HTML/CSS, no build step.

- `index.html` — main page (German)
- `impressum.html`, `datenschutz.html` — legal pages
- `styles.css` — all styling
- Deployed via GitHub Pages (legacy build, `gh-pages` branch). No GitHub Actions.

Edit content directly in the HTML. To make the Impressum fully real, fill the
`TODO Jens` placeholders (street, HRB number, USt-IdNr.).
